Les commandes à connaitre pour importer des bibiliothèques:
    1. import ... as ...
    2. from ... import ...

0. Préliminaires: installer une bibiliothèque sur python 

                Python a de nombreuses "bibliothèques" avec des types et elements prédéfinies.
                Mais pouvoir utiliser une bibliothèque il faut déjà l'avoir installé!

                Si vous aviez installé Python via une appli (le cas de 99% des gens):
                    - Suivez les instructions que vous donne votre appli


                Si vous aviez installé Python vous-même:
                Si vous avez un ordinateur Windows:
                    - ouvrez le command prompt (Appuyez le symbole "windows" + r, 
                    puis écrivez "cmd" dans la page qui s'ouvre)
                    - vérifiez sous quel nom python est installé:
                    
                          py --version
                          python --version
                          python3 --version
                    
                    - L'instruction qui vous donnera un résultat (qui n'est pas un message d'erreur)
                    correspond au nom qu'a Python dans votre ordinateur
                    - Vérifiez que pip est bien installé:
                    
                          nom_de_python -m pip --version
                    
                    - Si pip n'est pas installé alors:
                    
                         nom_de_python -m ensurepip --default-pip
                    
                    - Pour installer une biblithèque il suffit d'écrire:
                    
                         nom_de_python -m pip install nom_de_la_bibiliothèque
                    
                    Ex: 
                    
                         py -m pip install numpy 
                    
                
                Si vous avez un ordinateur Mac ou Linux:
                    - ouvrez le terminal (Appuyez le symbole "Command" + Espace, 
                    puis écrivez "Terminal" dans la page qui s'ouvre)
                    - vérifiez sous quel nom python est installé:
                    
                          py --version
                          python --version
                          python3 --version
                    
                    - L'instruction qui vous donnera un résultat (qui n'est pas un message d'erreur)
                    correspond au nom qu'a Python dans votre ordinateur
                    - Vérifiez que pip est bien installé:
                    
                          nom_de_python -m pip --version
                    
                    - Si pip n'est pas installé alors:
                    
                         nom_de_python -m ensurepip --default-pip
                    
                    - Pour installer une biblithèque il suffit d'écrire:
                    
                         nom_de_python -m pip install nom_de_la_bibiliothèque
                    
                    Ex: 
                    
                         py -m pip install numpy

1. import ... as ... 

                Pour utiliser une bibiliothèque il faut l'importer:
                
                    import nom_de_la_bibiliothèque
                    # Ex:
                    import numpy
                
                On peut ensuite librement utiliser les elements définies dans 
                cette bibiliothèque.

                Cependant il faudra TOUJOURS préciser de quelle bibiliothèque 
                vient notre element.

                Pour préciser la bibiliothèque d'où vient la element, 
                On écrit: nom_de_la_bibiliothèque.element_utilisée

                Par exemple le code qui suit ne marchera pas:
                
                    import numpy 
                    a = array([[1,0], [0,1]])
                 
                Alors que celui-ci fonctionne:
                
                    import numpy 
                    a = numpy.array([[1,0], [0,1]])
                 

                Certaines bibiliothèques ont des sous-bibliothèques.
                Pour les importer il suffit d'ajouter un ' . ' derrière le nom
                de la bibliothèque, suivi du nom de la sous bibiliothèque
                Par exemple:
                
                    import numpy.random
                    a = numpy.random.randint(1, 10)
                    print(a)
                 

                Les sous-librairies à connaitre sont:
                    - pyplot (qui est une sous-bibliothèques de matplotlib)
                    - linalg (qui est une sous-bibliothèques de numpy)
                    - random (qui est une sous-bibliothèques de numpy)
                Exemple:
                
                    import matplotlib.pyplot 
                    import numpy.linalg 
                    import numpy.random 
                


                On peut aussi attribuer un "alias" à la bibiliothèque qu'on importe 
                pour perdre moins de temps à écrire le nom de la bibiliothèque quand
                on l'utilise.
                
                Pour cela on ajoute "as un_alias_choisi" derrière le nom_de_la_bibiliothèque
                qu'on importe 

                Par exemple:
                
                    import numpy as np 
                    a = np.array([[1,0], [0,1]])
                

                On peut aussi donner à un alias à une sous-bibliothèque:
                
                    import numpy.random as rd 
                    a = rd.randint(0, 10)
                    print(a)
                

                Les alias classiques sont:
                    - al pour numpy.linalg 
                    - plt pour matplotlib.pyplot 
                    - rd pour numpy.random 
                    - np pour numpy 
                    - pd pour pandas

2. from ... import ... 

            On peut aussi chercher à importer seulement des éléments spécifiques
            d'une bibliothèque/sous-bibliothèque
            On écrit alors: 
            
                from nom_de_la_bibiliothèque import nom_de_element
                # Ex:
                from numpy import pi 
                from numpy import array
            
            On peut importer plusieurs élements en même-temps en mettant des virgules:
            
                from nom_de_la_bibiliothèque import element_1, element_2
                # Ex:
                from numpy import pi, e, array
            
            Lorsqu'on importe des elements directement on a pas besoin de préciser 
            de quelles bibiliothèque/sous-bibli elles viennent quand on les utilise
            
                from numpy import array
                a = array([[1,0],[0,1]])
                # On a pas besoin de mettre numpy.array dans ce cas.
                print(type(a))
            
            Alors que d'habitude on aurait: 
            
                # on ne précise pas ce qu'on importe, on dit juste d'utiliser la bibiliothèque
                import numpy 
                # on doit donc préciser de quelle librairie vient array
                a = numpy.array([[1,0],[0,1]]) 
                print(type(a))
            
            On peut aussi importer TOUT ce qu'il y a dans une bibiliothèque 
            Pour cela on dit qu'on importe " * " (l'étoile signifiant tout)
            
                from numpy import * 
                # On a pas besoin de dire que array vient de numpy
                # vu que * est équivalent à manuellement importer tout explicitement
                a = array([[1,0],[0,1]]) 
                print(type(a))
