Les VARAD dans Python

En général on peut simuler une VARAD de 2 manières différentes:
- utiliser directement la fonction appropriée apparentant à la bibliothèque numpy.random
- construire une fonction qui calcule la densité pour simuler nous-même la VARAD

Importons les bibiliothèques dont on aura besoin 

            
import numpy as np
import numpy.random as rd
import matplotlib.pyplot as plt

Faisons la représentation graphique d'une normale centrée réduite: 

            
    
            #En utilisant la bibliothèque numpy.random:
mu, sigma = 0, 1 # espérance et écart-type
simul = rd.normal(mu, sigma, 10000) # Fait une liste avec le résultat de 10000 normales
plt.hist(simul, 100, density=True) # fait un histogramme d'une densité de probabiloté, le 100 permet de modifier la précision je crois


            #En faisant tout soi-même:
def normale(m, sigma, x): #on construit nous-même la densité de probabilité 
    return 1/(sigma * np.sqrt(2 * np.pi)) * np.exp( - (x - m)**2 / (2 * sigma**2) )
# on peut utiliser cette fonction pour simuler nous-même la loi  et ne pas dépendre de np.random 

x = np.arange(-5, 5, 0.001) # l'axe des abscisses 
plt.plot(x, normale(mu, sigma, x)) # on peut dessiner manuellement la courbe 


plt.show() #affiche l'histogramme et la courbe

Faisons de même pour une exponentielle: 

            
           
            # Avec les fonctions données dans numpy.random:

l = 1 # Attention!! "lambda" est un mot-clef déjà utilisé par Python pour un certain type de fonction donc on ne peut pas nommer une variable "lambda"
simul = rd.exponential(l, 10000) 
plt.hist(simul, 100, density=True)

            # Et si on le fait nous-même

def expo(λ, x):
    return λ*np.exp(-λ*x)

x = np.arange(0, 5, 0.01) # l'axe des abscisses 
plt.plot(x, expo(l, x)) 

plt.show()
